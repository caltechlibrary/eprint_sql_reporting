# NAME

rpt-related_urls.bash

rpt-related_urls.bash

# DESCRIPTION

Generate a tab delimited file that reports all the eprintid and
their related url.

# EXAMPLE

~~~
    rpt-related_urls.bash > eprint_related_urls.tsv
~~~
